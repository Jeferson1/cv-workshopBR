CURRÍCULO DE 2 HORAS

Introdução ao HTML e CSS pela Ironhack

1. AMBIENTE

  1. Iniciar o Visual Studio Code
  2. Criar uma pasta com o nome CV no seu desktop.
  3. Arrastar a pasta criada e colocar dentro da janela do VS Code.
  4. Agora, dentro do VS Code: Na janela da esquerda
    • Clique com o botão direito e então, clique em novo arquivo
    • Nomeie ela index.html e clique Enter
    • Repita o processo e salve o novo arquivo como style.css
    • Clique com o botão direito e então clique em New Folder
    • Nomeie ela como images e aperte Enter
  5. Abra o seu navegador preferido (Chrome, Safari, Firefox...)
  6. Arraste o seu arquivo index.html para o navegador
  7. Estamos prontos!

5. Tags de nomeação - Classes do Bootstrap

O Bootstrap tem classificações especiais para o Grid system. Vamos usar algumas delas agora.

  • Para block#1, aplique as classes na tag div: block-1 col-lg-8
  • Para a primeira tag p em block#1, aplique a classe high-text
  • Para o block#2, aplique as seguintes classes à tag div: block-2 col-md-4
  • Para o block#3, aplique as seguintes classes na tag div: block-3 col-md-8
  • Para o block#4, aplique na tag div: block-4 col-lg-6
  • Para block#5, na div, aplique as seguintes classes: block-5 col-lg-6

Próximo passo, vamos integrar mais um pouco...

  1. Integre os blocks #1, #2, #3 dentro de uma div aplique as classes seguintes à ele: row row-1
  2. Integre os #4 and #5 dentro de uma div e aplique as seguintes classes: row m-0 mt-4
  3. Finalmente, integre todos os códigos dentro do body com uma div e aplique a classe container

7. Deploy!

É hora de colocar o currículo online!

  1. Crie uma pasta no google drive e compartilhe-a
  2. Selecione a opção "qualquer um na internet pode achar e ver"
  3. Acesse DriveToWeb e escolha a opção Google Drive
  4. Faça o login com a sua conta e permita o acesso aos seus arquivos (caso se sinta desconfortável com isso, crie uma conta Gmail apenas para armazenar seus arquivos públicos)
  5. Já está online! No painel de administração, selecione o link que corresponde ao seu CV e é isso. :)
